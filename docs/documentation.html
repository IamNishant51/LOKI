System Documentation

Technical reference for the Local Omni Knowledge Interface (LOKI) runtime environment.

01 Introduction

LOKI is a local-first AI agent wrapper around Ollama. It bypasses cloud latency and privacy risks by executing LLM inference directly on user hardware. It features a custom context-mapping engine that recursively indexes codebases for higher-fidelity RAG (Retrieval-Augmented Generation).

02 Installation

Step A: Runtime Dependencies

Install Ollama daemon and pull the target model.

curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3
Step B: LOKI Core

Install via NPM registry.

npm install -g @react-frameui/loki-ai

03 Usage Patterns

Interactive TUI
loki

Launches the ncurses-like terminal interface.

Quick Query
loki chat "How do I reverse a binary tree?"

One-shot inference command.

System Diagnostic
loki doctor

Verifies Ollama connectivity and model integrity.

04 WhatsApp Bridge

LOKI can spawn a headless WhatsApp Web instance to serve as a remote control.

Security Note: All messages are processed locally. No data is sent to Meta servers other than the encrypted message transport itself. 

loki server

Scan the generated QR code with your mobile device to link.
