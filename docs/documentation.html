Documentation

Complete guide to using LOKI - your privacy-first local AI assistant.

Introduction

LOKI (Local Omni Knowledge Interface) is a powerful AI assistant that runs entirely on your machine. Unlike cloud-based AI services, LOKI processes everything locally using Ollama, ensuring your code and conversations never leave your computer.

Key Features:

  • 100% local processing with Ollama
  • Interactive Terminal User Interface (TUI)
  • WhatsApp integration for mobile access
  • VS Code extension for in-editor assistance
  • Built-in tools for file system, git, math, and more
  • Conversation memory and context awareness

Installation

Prerequisites

Before installing LOKI, you need to have the following:

  • Node.js (v18 or higher) - Download here
  • Ollama - Download here

Step 1: Install Ollama

Download and install Ollama from ollama.ai, then pull a model:

ollama pull llama3

You can also use other models like mistral, codellama, or phi.

Step 2: Install LOKI

Install LOKI globally via npm:

npm install -g @react-frameui/loki-ai

Step 3: Verify Installation

Check that LOKI is installed correctly:

loki --version

Usage

Interactive TUI

Launch the interactive menu by running:

loki

This opens the main menu where you can:

  • Chat with LOKI
  • Run agent workflows
  • Index your repository
  • Connect WhatsApp
  • Run system diagnostics

Direct Chat

Start a chat session directly:

loki chat

Or ask a quick question:

loki chat "what time is it?"

Commands

loki chat [message]

Start an interactive chat session or send a single message.

loki chat loki chat "explain this code" loki chat --agent explain loki chat --no-memory

Options:

  • -a, --agent <name> - Choose agent: dev, explain, refactor (default: dev)
  • --no-memory - Disable conversation memory
  • --no-stream - Disable streaming output
  • -f, --file <path> - Add file to context
  • -d, --dir <path> - Add directory to context

loki doctor

Run system health checks to verify Ollama connection and configuration.

loki doctor

loki index

Index your current repository for better context awareness.

loki index

loki server

Start the WhatsApp integration server.

loki server

WhatsApp Integration

LOKI can integrate with WhatsApp, allowing you to chat with your AI assistant from your phone.

Setup

  1. Start the WhatsApp server:
    loki server
  2. A QR code will appear in your terminal
  3. Open WhatsApp on your phone → Settings → Linked Devices → Link a Device
  4. Scan the QR code
  5. Start chatting with LOKI!

Note: The WhatsApp integration runs a local server. Your messages are processed on your machine and never sent to external servers.

Usage

Once connected, you can message LOKI directly on WhatsApp:

  • Ask questions: "What time is it?"
  • Get help: "How do I use git rebase?"
  • Quick calculations: "What's 15% of 250?"

VS Code Extension

The LOKI VS Code extension brings AI assistance directly into your editor.

Installation

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
  3. Search for "LOKI AI Assistant"
  4. Click Install

Or install from the VS Code Marketplace.

Features

  • Explain Code: Right-click on selected code → "LOKI: Explain Code Selection"
  • Chat: Open command palette (Ctrl+Shift+P) → "LOKI: Open Chat"

Available Tools

LOKI has built-in tools that it can use to help you:

Time & Date

  • get_time - Get current time
  • get_date - Get current date

Math

  • calculate - Evaluate math expressions

File System

  • list_files - List files in a directory
  • read_file - Read file contents

Git

  • git_status - Get git status
  • git_log - Get recent commits
  • git_diff - Get git diff

Configuration

LOKI can be configured using environment variables. Create a .env file in your project root:

# Ollama Configuration OLLAMA_HOST=http://localhost:11434 OLLAMA_MODEL=llama3 # Optional: Groq API (for cloud fallback) GROQ_API_KEY=your_api_key_here

Environment Variables

  • OLLAMA_HOST - Ollama server URL (default: http://localhost:11434)
  • OLLAMA_MODEL - Model to use (default: llama3)
  • GROQ_API_KEY - Optional Groq API key for cloud fallback

Troubleshooting

LOKI doesn't respond

Check if Ollama is running:

ollama list

If Ollama isn't running, start it and ensure you have a model pulled:

ollama pull llama3

Connection refused error

Make sure Ollama is running on the correct port (default: 11434). Run the doctor command:

loki doctor

WhatsApp QR code not appearing

Ensure you have the required dependencies installed. Try reinstalling:

npm install -g @react-frameui/loki-ai

VS Code extension not working

Make sure:

  • LOKI CLI is installed globally
  • Ollama is running
  • You've reloaded VS Code after installation
